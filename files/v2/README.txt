The v2 version of the IBM Aspera Connect SDK is extremely outdated and should NOT be
used. It will be completely removed in a future release. Please update your web 
applications to use the v4 version of the SDK instead.

For example:
`https://d3gcli72yxqn2z.cloudfront.net/connect/v4/asperaweb-4.min.js`
`https://d3gcli72yxqn2z.cloudfront.net/connect/v4/connectinstaller-4.min.js`

See https://developer.asperasoft.com/web/connect-client/hosting for more
information.
