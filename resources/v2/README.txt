the SDK package. Please update your web applications to use the v4 version of the SDK instead.

For example:
`https://d3gcli72yxqn2z.cloudfront.net/connect/v4/asperaweb-4.min.js`

See https://developer.ibm.com/apis/catalog/aspera--aspera-connect-sdk/Introduction for more
information.
